# ITA-Projekt: Farbenlehre-Website (mit SASS & Grid)

Hallo! Dies ist mein zweites grosses Web-Projekt aus meiner ITA-Ausbildung (2. Jahr).

Das ursprungliche Ziel war eine Praesentation ueber Farbenlehre. Ich habe mich entschieden, das Thema stattdessen als interaktive Website umzusetzen, um meine Web-Development-Faehigkeiten zu vertiefen.

**WICHTIG:** Sie koennen sich das Projekt hier live ansehen:
**[Live-Demo (geoeffnet mit GitHub Pages)](https://DEIN-BENUTZERNAME.github.io/ita-projekt-farbenlehre-website)**

## Technischer Fokus & Fortschritt

Dieses Projekt war fuer mich ein wichtiger naechster Schritt nach den Grundlagen der ersten Website.

Der technische Fokus lag darauf, meine CSS-Faehigkeiten auf ein professionelleres Niveau zu heben. Ich habe hier zum ersten Mal **SASS (einen CSS-Praeprozessor)** fuer die Organisation der Styles und das **CSS Grid System** fuer das Seitenlayout verwendet.

Der SASS-Code ist sauber in Partials aufgeteilt, um die Struktur uebersichtlich zu halten.

## Projekt-Kontext (Gruppenarbeit)

Obwohl dies offiziell als Gruppenprojekt eingetragen war, habe ich die vollstaendige Konzeption, das Design und die gesamte technische Umsetzung (HTML, SASS, Grid) selbststaendig durchgefuehrt.

## Hinweis

Genau wie mein erstes Projekt lag auch hier der Fokus auf der Desktop-Ansicht, um die neuen Layout-Techniken zu erlernen. Die Seite ist daher **nicht** fuer mobile Endgeraete optimiert (nicht responsiv).
